CF 1731D. Valiant’s New Map

Finally a question I fully solved myself! (1700 too lol)

The question is for each test case (1≤t≤1000)(1 \leq t \leq 1000)(1≤t≤1000), you want to find the maximum lll such that
you can find a l×ll \times ll×l square in a n⋅m (1≤n,m≤200)n \cdot m \space (1 \leq n, m \leq 200)n⋅m (1≤n,m≤200) grid where every element is greater or equal to lll.

Well, a easy idea is to binary search lll, enumerate through every square and do 2D range min queries!
This is just a basic binary search + 2D range min query right? 2D sparse table lets gooooo!
Funnily enough, 2D ST is actually a solution that was given in the tutorial (albeit a bit overkill),
and 2D segtree actually wouldn’t pass due to O(logn+logm)O(logn+logm)O(logn+logm) each query.

Lets try to solve it in an elegant manner instead of overkilling with weird data structures!
Of course when talking about 2D queries, you would recall that you could do 2D prefix arrays.
But thats for 2D range sum queries, how to we transform the question into checking range sums?
Notice that if we let arr[i]=(arr[i]>=l)arr[i] = (arr[i] >= l)arr[i]=(arr[i]>=l), we would get a array with only zeros and ones,
then if we do prefix sum, a query that equals to l×ll \times ll×l means that every element inside is greater or equal to lll!

CF 1114C. Trailing Loves (or L’oeufs?)

This problem is super hard for me… pure math is just death for me XD

The problem is to find the number of trailing zero digits when representing n! (1≤n≤1018)n! \space (1 \leq n \leq 10^{18})n! (1≤n≤1018) in base b (2≤b≤1012)b \space (2 \leq b \leq 10^{12})b (2≤b≤1012).
It is also equivalent to finding the max rrr such that n!n!n! is divisible by brb^rbr. (Makes alot of sense if you think about it).
Well, the observations above are still pretty reasonable, but n!n!n! is a super big number! how do we do this?
Apparently, there is an exact formula dealing with this problem: Legendre’s Formula
The formula states that the largest rrr such that n!n!n! is divisible by pnp^npn is ∑i−1∞⌊npi⌋\displaystyle \sum_{i-1}^{\infty} \lfloor \frac{n}{p^i} \rfloori−1∑∞​⌊pin​⌋.
I was confused by this formula at first, but it made sense after some thinking:
n!=1⋅2⋅...nn! = 1 \cdot 2 \cdot ... nn!=1⋅2⋅...n, how many numbers between 111 to nnn are divisible by ppp? ⌊np⌋\lfloor \frac{n}{p} \rfloor⌊pn​⌋ numbers.
Now we counted the numbers divisible by ppp, but we only counted the numbers divisible by p2p^2p2 once when they should contribute twice. So what we do is do this for every power of ppp and sum up the values to get what we want.
The formula requires the base bbb to be prime though, while we have any number in [2,1012][2, 10^{12}][2,1012].
What we can do is to prime factorize bbb as p1x1⋅p2x2⋅...pkxkp_1^{x_1} \cdot p_2^{x_2} \cdot ... p_k^{x_k}p1x1​​⋅p2x2​​⋅...pkxk​​ and apply the formula with every prime factor pip_ipi​ to obtain epi(n!)e_{p_i}(n!)epi​​(n!).
Finally, we can get the answer for bbb by min(yixi)\displaystyle \text{min}(\frac{y_i}{x_i})min(xi​yi​​).
